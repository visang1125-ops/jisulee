# 데이터 디렉토리

이 디렉토리에 예산 데이터 엑셀 파일을 배치하세요.

## 파일명

- **`budget.xlsx`** - 서버 시작 시 자동으로 로드되는 예산 데이터 파일

## 엑셀 파일 형식

엑셀 파일은 다음 컬럼을 포함해야 합니다:

| 컬럼명 | 설명 | 필수 | 예시 |
|--------|------|------|------|
| 부서 | 부서명 | ✅ | DX전략 Core Group |
| 계정과목 | 계정과목 | ✅ | 광고선전비(이벤트) |
| 월 | 월 (1-12) | ✅ | 1 |
| 연도 | 연도 | ✅ | 2025 |
| 구분 | 예산 또는 실제 | ✅ | 예산 |
| 금액 | 금액 | ✅ | 10000000 |
| 프로젝트명 | 프로젝트명/세부항목 | ✅ | 이벤트프로모션 |
| 산정근거/집행내역 | 산정근거 또는 집행내역 | ✅ | 2025년 1분기 키즈 이벤트 프로모션 집행 |
| 예산 내/외 | 예산 내 또는 예산 외 | ❌ | 예산 내 |
| 사업구분 | 키즈, 초등, 중등, 전체 | ❌ | 키즈 |
| 고정비/변동비 | 고정비 또는 변동비 | ❌ | 변동비 |

## 사용 방법

1. 엑셀 파일을 준비합니다 (위 형식에 맞춰서)
2. 파일명을 `budget.xlsx`로 저장합니다
3. 이 디렉토리(`data/`)에 배치합니다
4. 서버를 재시작하면 자동으로 로드됩니다

## 주의사항

- 엑셀 파일이 없으면 샘플 데이터가 자동으로 생성됩니다
- 서버 재시작 없이 데이터를 다시 로드하려면 API를 사용하세요:
  - `DELETE /api/budget/clear` - 모든 데이터 삭제
  - `POST /api/budget/reload-from-excel` - 엑셀 파일에서 다시 로드
